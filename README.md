# Javascript TDD   
Programming kit to practice TDD (Test Driven Development)    
### Tools include :   
 - Nodejs
 - Jest (test runner)
 - eslint
 - babel  

To Run
```
	> git clone https://github.com/jslnriot/javascript-tdd.git
	> cd javascript-tdd
	> yarn  (or npm install)
	> yarn test OR npm test (to run the test suite)
```
